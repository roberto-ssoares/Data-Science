## Arquivo `src/utils_timeseries.py`

Este módulo vai concentrar funções de:

- carregamento e preparo da série temporal

- garantia de índice datetime

- preenchimento de valores faltantes

- diferenciação da série

- teste de estacionariedade (ADF + relatório)

## Arquivo `src/plot_functions.py`

Este módulo vai centralizar os plots mais usados:

- plot único da série

- decomposição (usando `seasonal_decompose`)

- rolling statistics

- ACF/PACF em conjunto
